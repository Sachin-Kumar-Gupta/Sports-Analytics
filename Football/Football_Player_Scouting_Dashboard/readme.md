# Football Player Scouting Dashboard

An interactive scouting dashboard for football coaches and selectors to analyze players based on performance statistics, positions, and similarity metrics.

---

## Features

- **Player Radar Chart**: Visualize key stats per player.
- **Similar Players**: Find top 5 similar players based on selected metrics.
- **Position Filter**: Filter players by position.
- **Cluster Analysis**: Explore KMeans-based player clusters.
- **Interactive Plots**: Built using Plotly for hover info and multiple player comparisons.

## Installation

1. Clone this repository:

git clone https://github.com/<your-username>/football-scouting-dashboard.git
cd football-scouting-dashboard

2. Install dependencies:

pip install -r requirements.txt

3. Run the dashboard:

streamlit run app.py
